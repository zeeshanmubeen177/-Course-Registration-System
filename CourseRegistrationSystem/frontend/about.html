About the Project

Open Source Software Development — Final Semester Project

Problem Statement

In many educational institutions, course registration is performed manually or through outdated systems. This is time-consuming, error-prone and hard to manage at scale. Students struggle to view available courses, register, and track their enrollment, while administrators find it difficult to maintain records. This system automates the entire process through a clean web interface backed by a reliable database.

Objective

Build a full-stack web application that allows students to register for courses online and enables administrators to manage courses and registrations efficiently, with all data stored in a centralized PostgreSQL database.

Target Users

Students — register, log in, browse courses, enrol, drop, and view their schedule.
Administrators — manage student records, add/update/delete courses, and monitor all registrations and statistics.
